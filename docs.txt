## ğŸ§° Your First FastAPI App
```python
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
async def read_root():
    return {"message": "Hello, FastAPI!"}
```
Run it with:
```bash
uvicorn main:app --reload
```

Example: `/items/?skip=5&limit=20`

## ğŸ“¥ Request Body (POST)
```python
from pydantic import BaseModel

class Item(BaseModel):
    name: str
    price: float
    is_offer: bool = False

@app.post("/items/")
async def create_item(item: Item):
    return item
```
Example request body:
```json
{
  "name": "Laptop",
  "price": 999.99,
  "is_offer": true
}
```

## ğŸ” Dependency Injection
Reusable logic like authentication or database access is easy:
```python
from fastapi import Depends

async def common_parameters(q: str = None, skip: int = 0, limit: int = 10):
    return {"q": q, "skip": skip, "limit": limit}

@app.get("/items/")
async def read_items(commons: dict = Depends(common_parameters)):
    return commons
```

## ğŸ”’ Security (OAuth2, JWT)
FastAPI supports OAuth2 and JWT for authentication:
```python
from fastapi import Depends
from fastapi.security import OAuth2PasswordBearer

oauth2_scheme = OAuth2PasswordBearer(tokenUrl="token")

@app.get("/users/me")
async def read_users_me(token: str = Depends(oauth2_scheme)):
    return {"token": token}
```
